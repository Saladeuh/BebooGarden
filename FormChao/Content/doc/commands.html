Commands

Main Commands

  • Move: WASD or the arrow keys
  • State the condition of the bbeboo(s): F
  • State the number of fruits in the basket: G
  • State the number of tickets: T
  • Shake/pet: hold Enter while alternating between the up and down arrow keys or W and S
  • Whistle, interact with an object: Space
  • Call a beboo: voice or numbers
  • Put an object in the bag: Enter
  • Enter a new location: Enter

Volume Control

  • Lower the global volume: F2
  • Raise the global volume: F3
  • Lower the music volume: F5
  • Raise the music volume: F6
  • Mute the music: F4

In the Menus

  • Move: the arrow keys/WASD
  • Confirm: Enter/Space
  • Back: Escape
